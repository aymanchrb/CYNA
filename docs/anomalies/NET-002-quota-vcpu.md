# NET-002 — Quota Azure limité à trois VM

## Description
L'abonnement Azure for Students limite le nombre de vCPU disponibles, empêchant le déploiement de toutes les VM prévues.

## Sévérité
Haute

## Résolution
Consolidation des services : Web+APP sur une seule VM et PostgreSQL+Wazuh sur une autre.

## Statut
Résolu
