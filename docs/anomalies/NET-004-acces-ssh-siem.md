# NET-004 — VM Data/SIEM sans IP publique inaccessible directement

## Description
La VM Data/SIEM ne possède pas d'IP publique, la rendant inaccessible directement depuis l'extérieur.

## Sévérité
Normal

## Résolution
Utilisation de la VM WebApp comme machine de rebond SSH vers la VM Data/SIEM.

## Statut
Résolu
