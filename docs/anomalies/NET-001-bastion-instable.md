# NET-001 — Bastion instable et connexion RDP impossible

## Description
Le bastion Azure s'est révélé instable, rendant la connexion RDP impossible.

## Sévérité
Moyenne

## Résolution
Remplacement temporaire par une IP publique avec RDP limité à l'IP administrateur (92.129.221.171/32).

Résolu
